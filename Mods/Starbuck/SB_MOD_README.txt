This is Starbuck's content mod for Vren's Lab Rats 2. The goal of this mod is to expand the Lab Rats 2 game with additional content and options for players to enjoy. Lab Rats 2 is a work in progress, and so is this mod. There are likely errors, bugs, typos, etc, so install at your own risk.



	**********Section 1: Version**************
Starbuck's content mod follows Vrens Lab Rats version numbering. If multiple version of this mod are needed for a given Lab Rats 2 version number, I will include lettering in brackets following the version number. For example, if Lab Rats 2 is on version 1.0, the first release of this mod will be 1.0, followed by 1.0[a], etc.

The current version of this mod is 0.15.0



	**********Section 2: Installation ************
Copy the contents of this zip file to the game folder location on your hard drive where Lab Rats 2 is. For example, it may be similar to:

C:\Downloads\Lab_Rats_2-v0.15.1-pc\game\

*NOTE*
You may have to start a new game for this mod to work correctly, but it SHOULD work with existing save games.
